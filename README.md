# Temple & Pilgrimage Crowd Management System

A MERN stack application for managing crowd flow, bookings, and emergency services at pilgrimage sites.

## Features
- **Pilgrims**: Book slots, View Tickets, SOS Alert.
- **Management**: Monitor Crowd, Configure Slots.
- **Police**: Heatmaps, Incident Reporting.
- **Medical**: Resource Tracking.

## Tech Stack
- MongoDB Atlas
- Express.js
- React (Vite) + Tailwind CSS
- Node.js
- Socket.io (Real-time updates)

## Setup
1. **Server**:
   ```bash
   cd server
   npm install
   # Configure .env
   npm start
   ```

2. **Client**:
   ```bash
   cd client
   npm install
   npm run dev
   ```
